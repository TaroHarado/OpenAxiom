Features

Built for speed.
Wired for safety.

Solana + Robinhood Chain

Solana swaps route through Jupiter and Pump.fun. Robinhood Chain trades execute via Uniswap V3 with automatic pool selection and slippage guard.

Local signing

Use your browser wallet, or import a hot key — it is device-encrypted and decrypted only within the active extension session. Nothing is sent anywhere.

Public RPC by default

Ships pre-configured with PublicNode and dRPC — free, reliable, no keys required. Swap in Helius or Shyft any time for extra throughput.

Presets & hotkeys

Scalp / Standard / Ape strategies with dedicated slippage, priority fee and buy amounts. Keyboard shortcuts map to every buy/sell button.

Local PnL & history

Cost-basis and realized PnL are rebuilt from on-chain history and stored in Chrome local storage. Your trade log never leaves the browser.

No custody · no middleman

Trench holds no funds and takes 0% on every trade — no default fee, no optional fee, no treasury cut. Your transactions go straight to your own RPC.

Install

Running in about
two minutes.

Loaded as an unpacked extension directly into Chromium — Chrome, Brave, Edge. No store approval, no gatekeeper.

01

Clone & build

Requires Node.js 18+. Clone the repository and build the dist/ folder.

# clone the repo
git clone https://github.com/TaroHarado/Trench.git
cd trench
npm install
npm run build
02

Load into your browser

Open chrome://extensions, enable Developer mode in the top-right corner, click Load unpacked, and select the generated dist/ folder.

03

Open a market

Navigate to gmgn.ai/?chain=robinhood or axiom.trade/pulse. The overlay mounts automatically, and BUY buttons appear on every card. Pin the extension icon for quick popup access.

04

Add a wallet (optional)

Browse freely with no keys. To trade, connect your browser wallet — or import a hot private key in Options. It is encrypted on your device and never transmitted.

FAQ

Good questions.

Is it really free? +

Yes. The extension is MIT-licensed and takes 0% on every trade — there is no default fee and no optional fee. It runs on free public RPC endpoints, so there is nothing to pay and nothing to sign up for.

Where are my private keys stored? +

On your device only. Hot keys are encrypted with a device key and decrypted only for the active extension session. They are never sent to any server. You can also use an existing browser wallet and skip key import entirely.

Which chains and DEXes are supported? +

Solana via Jupiter and Pump.fun, and Robinhood Chain via Uniswap V3 with automatic pool and route discovery. The overlay works on gmgn.ai and Axiom.

Do I need a paid RPC? +

No. The extension ships with PublicNode and dRPC configured by default — both are free and require no API key. You can paste a Helius, Shyft, or QuickNode endpoint in Options at any time for lower latency.

Does it work on other sites besides gmgn.ai? +

Yes. The floating overlay also mounts on axiom.trade. Additional platforms can be added by updating the content script match patterns in the extension manifest.

Is this financial advice? +

No. Trench is a tool. Trading new tokens is high-risk and you can lose everything. Always verify what you sign. You are fully responsible for your own trades.
